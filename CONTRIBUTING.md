# Contributing to DataLens

Thank you for your interest! Fork the repo, create a feature branch, make your changes, run tests, and open a PR.

## High-priority contributions
- New database connectors (MySQL, BigQuery, Snowflake)
- Additional chart types  
- Dashboard export (PDF, PNG, HTML)
- Test coverage improvements

## Code Style
- Python: Follow PEP 8, use type hints
- TypeScript/React: Functional components only

## Commit Message Format
- `feat:` — new feature
- `fix:` — bug fix
- `docs:` — documentation only
- `test:` — adding tests
